# 🌐 Translate +

A modern real-time translation web application built with **React**, **Redux Toolkit**, and **TailwindCSS**. Users can translate text between 156 languages, auto-detect the source language, listen to translations via speech synthesis, and review recent translations — all powered by the **Deep Translate API** via **RapidAPI**.

---

## 🚀 Features 



🌍**Language Selection** – Choose source and target languages from 156 supported options via searchable dropdowns.   
🔍**Auto-Detect Language** – Automatically detect the source language when translating.  
🔄**Swap Languages** – Instantly swap source and target languages along with their text.  
✍️**Text Translation** – Translate text with a side-by-side input and output layout.   
🔊**Speech Synthesis** – Listen to both source and translated text using the Web Speech API.  
📋**Copy Translation** – Copy the translated text to clipboard with one click.   
🧹**Clear Input** – Reset the source text area quickly.  
📜**Translation History** – View recent translations with language pair, date, and text preview.  
⚡**Redux Toolkit** – Centralized state management with `createAsyncThunk` for async API calls. 
📱**Fully Responsive** – Clean, dark-themed layout optimized for all screen sizes.
